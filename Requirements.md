### Basic implementation of a CMS using Ruby on Rails.

#### Cienfuegos CMS v0.1.0

**Features:**

- A platform for simple blogging.
- Use Bootstrap 3.
- Create a user system.
- Every user have the same html template.
- Users can create a new post using any wysiwyg library.
- Users can edit previous created content.
- The app have a global simple administration section (PA).
- Registered users report in the PA.
- Admins can edit the user content in the PA.
- The app must be created using TDD (Bonus)
- The code must be well documented and well formatted (Bonus)
